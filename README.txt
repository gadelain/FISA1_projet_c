Le Programme de Cryptage-Décryptage permet, comme son nom l'indique de crypter et de décrypter un texte selon plusieurs méthodes de cryptages/decryptages.
Dans un premier temps l'utilisateur a le choix entre crypter un texte ou decrypter un texte precedement crypté. L'utilisateur a le choix entre saisir sont propre texte 
directement dans le programme en allant dans le menu ANNEXE ou alors utiliser un texte au format .txt preexistant. 

Dans un second temps l'utilisateur à le choix entre 3 methodes différentes de cryptage/decryptage. Ces 3 méthodes sont classées en fonction de leurs niveaux de 
complexité de cryptage/decryptage, allant de 1, pour la méthode la moins sécurisée, à 3 pour la méthode la plus sur.

- Le premier niveau de complexité correspond à la méthode du chiffre de Cesar.


- Le deuxieme niveau de complexité correspond au cryptage par Clef.


- Le troisieme niveau de complexité correspond a la méthode du cryptage par image.


Pour decryper un texte, il faut utiliser la même méthode que lors du cryptage.

A la fin de chaque cryptage/decryptage, l'utilisateur a la possibilité de recuperer le texte crypté ou décrypté dans le main.
